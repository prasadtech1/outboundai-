🔧 Setup Guide
One-Time Setup Steps
  1. Run supabase_schema.sql in Supabase SQL Editor
  2. Go to ⚙️ Settings → fill in LiveKit, Gemini, Vobiz → Save
  3. Click ⚡ Create SIP Trunk in LiveKit
  4. Go to ✏️ AI Prompt → customise for your business
  5. Go to 🤖 Agents → create at least one profile
  6. Go to 📞 Single Call → test with your number
  7. Hear AI speak within 5–10s ✅
